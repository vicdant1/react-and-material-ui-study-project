# Running the project

 1. Download the source code (believe me, you will need it)
 2. Open Windows CMD on the directory you've downloaded it
 3. Run this commands there:
	 1. `npm install`
	 2. `npm run backend` 
	 3. `npm start`
 4. It's done, enjoy the app!
